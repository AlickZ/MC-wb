McBOX
=====

Web manager for MineCraft Bukkit Servers.Use `Remote Toolkit` and `JSONAPI`.

requirements:

- Ruby 1.9.3/2.0.0
- MongoDB 2+
- *NIX or Microsoft Windows

***Make sure only `rack-1.4.1` alivable in your env..***

Only works with `Ruby 1.9.3` when using Windows.
